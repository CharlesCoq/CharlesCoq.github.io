H a r d S k i l l s

Css

CSS

Le CSS (Cascading Style Sheets) permet de créer des pages web à l'apparence soignée. Grâce aux CSS, nous pouvons contrôler exactement l'affichage de chaque élément HTML dans le navigateur et ainsi présenter vos documents avec la mise en forme de votre choix.

Html

HTML

HTML signifie "HyperText Markup Language". C’est un langage qui permet de composer des pages web. Son but est d’encadrer les différents éléments présents dans une page (images, titres, paragraphes ...) par des balises pour leur permettre d’être mis en forme secondairement

Php

PHP

PHP: Hypertext Preprocessor, est un langage de programmation libre, principalement utilisé pour produire des pages Web dynamiques via un serveur web.

JavaScript

JavaScript

JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives. Il est au cœur des langages utilisés par les développeurs web.

Bootstrap

Bootstrap

Bootstrap est une collection d'outils utiles à la création du design (graphisme, animation et interactions avec la page dans le navigateur, etc.) de sites et d'applications web.

WordPress

WordPress

WordPress est un système de gestion de contenu (SGC ou content management system (CMS) en anglais) gratuit, libre et open-source.

Figma

Figma

Figma est une plateforme collaborative pour éditer des graphiques vectoriels et faire du prototypage. Elle permet de concevoir des design systems pour faciliter la création de sites web et d’applications mobiles.

Notion

Notion

Notion est une application de prise de notes, de bases de données, des tableaux Kanban, de wikis, de calendriers et de rappels. C'est à la fois un idéateur et un wiki.

VSC

Visual Studio Code

Visual Studio Code est un éditeur de code extensible développé par Microsoft pour Windows, Linux et macOS.

My SQL

My SQL

My SQL est un système de gestion de bases de données relationnelles. Il est distribué sous une double licence GPL et propriétaire. Il fait partie des logiciels de gestion de base de données les plus utilisés au monde.

JMerise

JMerise

JMerise est un logiciel dédié à la modélisation des modèles conceptuels de donnée. Il permet les relations réflexives, la généralisation et la spécialisation des entités. Il génère le MLD et le script Mysql
